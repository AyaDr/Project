Projet académique utilisant l’apprentissage automatique pour détecter les réclamations frauduleuses en assurance auto.  
Jeu de données : Kaggle (15 420 réclamations, 6% fraude).  
Modèles testés : Régression Logistique, Forêt Aléatoire, Arbre de Décision, XGBoost, SVM, KNN, MLP.  
Résultat : **XGBoost** est le plus performant (Accuracy 0.92, Rappel 0.83, AUC 0.96).
